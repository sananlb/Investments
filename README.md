## Ключевые темы исследования

### Рынок ИИ
- Объём рынка: $240-270 млрд к 2026 году
- CAGR генеративного ИИ: 40-50%
- Потенциал добавления $13-22 трлн к мировой экономике к 2030

### AMD vs NVIDIA
- Доля рынка: NVIDIA 80-92%, AMD 5-8%
- Ключевое различие: экосистема CUDA (18 лет) vs ROCm (10 лет)
- Партнёрство AMD + OpenAI: потенциал $100+ млрд

### Инвестиции
- OpenAI: оценка $157 млрд
- Anthropic: оценка $60 млрд
- xAI: оценка $50 млрд
- Корпоративные CAPEX на AI: $150+ млрд ежегодно
