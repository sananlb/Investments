# Junior

Junior miners, smaller cyclicals and high-risk commodity optionality.

Группа в TradingView: `Junior`.

## Ключевые компании

- [GLN](GLN.md) - Glencore: large diversified miner/trader used as sector anchor.
- [ALB](ALB.md) - Albemarle: lithium cycle benchmark.
- [LAC](LAC.md) - Lithium Americas: lithium development optionality.
- [MP](MP.md) - MP Materials: rare earths and US strategic minerals.
- [NEO](NEO.md) - Neo Performance Materials: rare earths/materials exposure.
- [BEZ](BEZ.md) - Besra Gold: junior gold exposure.
- [PEI](PEI.md) - Pengana Private Equity: portfolio holding / private market exposure.
