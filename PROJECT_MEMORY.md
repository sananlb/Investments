# Память проекта: инвестиционная библиотека

## Назначение папки

Эта папка используется как рабочая инвестиционная библиотека. Цель проекта - помогать инвестору постоянно быть "в рынке": понимать ключевые темы, отслеживать портфель, связывать макро/сектора/компании между собой и поддерживать карту знаний для принятия инвестиционных решений.

Изначально папка была посвящена AI research, но дальше стала более широкой библиотекой исследований по инвестициям. Верхняя папка проекта называется `Инвестиции`; `AI` - одна из отраслевых папок внутри нее.

## Роль пользователя

Пользователь - частный инвестор. Ему важно:

- видеть общую картину рынка;
- понимать структурные темы на годы вперед;
- связывать идеи с текущим портфелем;
- хранить исследования по секторам и компаниям;
- фиксировать ошибки, решения и watchlist;
- регулярно обновлять картину рынка, а не делать разовые отчеты.

## Принцип карты знаний

Исследования должны связываться не только по компаниям, но и по тезисам:

```text
AI boom
-> compute demand
-> GPU / ASIC demand
-> HBM / CoWoS / networking bottlenecks
-> дата-центры
-> electricity / grid / cooling bottleneck
-> бенефициары: MRVL, AVGO, ALAB, VRT, ETN, CEG, VST, GE Vernova
-> связь с портфелем: MRVL, AMD
```

## Рабочее правило для Codex

При новых инвестиционных обсуждениях:

1. Сначала проверять, есть ли уже локальное исследование по теме.
2. Если вопрос связан с текущими позициями, перечитать Google Sheets `Портфель`.
3. Для текущих рыночных данных, цен, новостей, отчетов, мультипликаторов и прогнозов использовать актуальные источники.
4. Фиксировать важные выводы в библиотеке, если пользователь просит сохранить или если это явно становится частью долгосрочной карты знаний.
5. Разделять факты, гипотезы, риски и инвестиционные действия.
